BUILDING A RESPONSIVE WEBSITE

Summer 2013 | Individual Project

This was a personal project where I designed and built a fully responsive personal portfolio website from scratch in html, css and javascript. I made simple wireframes, iterated between designs, and coded the entire site in html and css from scratch.
I began by ideating different layouts for my website based on the content that I expected to be on the site. Once I had the different layouts drawn out, I wrote out pros and cons for each design, and narrowed down my choices.
My first iteration website was a fixed width website, with only one page. Once I finished this, I began a second iteration, to make the website responsive to all different display sizes. I learned a lot from the first iteration, from not only a design perspective, but also from a coding perspective. These insights helped me design the current version of the site.
My final website displays differently on each display. This shows how the website responds to change in display sizes.
